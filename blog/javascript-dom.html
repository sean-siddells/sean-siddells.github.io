HTML and CSS

HTML and CSS are two of the main coding languages used in building a website. The HTML is like the skeleton, it provides the structure and content of the website. The CSS is like the flesh of the website, provides information that then takes the barebones structure of the site and gives it color, formatting, and more specific information about spacing. The CSS handles the aesthetic side of the website and much like how we wouldn't be that nice to look at without flesh on our bodies, dresses the skeleton structure of the website so that it's easier on the eyes.

Control Flow

Control flow is the order in which a computer executes statements in a piece of code. Generally this would move from the top to bottom, executing the first the first written statement first. Control flow can be thought of like a three-course meal at a restaurant - we eat the meal in the set order that it is given to us; starting with an entree and finishing with a dessert. However there are many functions that can change the order in which it is executed, like loops for example. Loops repeat a task until the conditions set are returned with a false result. An example of this could be like an arcade game - The player continues to play the game until they make a mistake, where the game ends and the results are logged in the form of a high score.

The DOM

The Document Object Model or DOM is an interface that allows the user to interact and manipulate the elements (or nodes) of a web document. It is used to build and make changes to websites. We could used the DOM to change the text of a heading or add new style classes to apply to a paragraph. It's extremely helpful as it allows the user to click on the elements of the website and then see what line of code that it pertains to. It is a useful tool for fine tuning a website and making changes in a very interactive manner, where you can see the changes side by side with the DOM.

Accessing Data from Arrays and Objects

Arrays and Objects are similar in the sense that they are both variable types that can store collections of data. Objects are used when the data is a collection of "things" with characteristics for example it would look something like this:

var person {
  "name": "Sean",
  "age": 27,
  "hair colour": "Brown"
};

Objects can be accessed using dot or bracket notation. This looks like:

person.name
OR
person['name']

both of these would return the value "Sean".

An array is an ordered list of data within a single variable. for example:

var flatmates ['Kyren', 'Louis', 'Adam'];

Arrays use "zero-indexing" meaning that the reference number of the items in the array starts from zero. So in order to bring up the name 'Kyren' from the flatmates array we would use:

flatmates[0];

Functions

A function is a series of statements that execute a task or action. They require an input and only return an output when they are activated or 'called'. They behave as a special kind of object and therefore can be treated like one when writing code. They are useful for performing calculations, returning values and so on without having to write long lines of code over and over again. This keeps the code succinct and the expressed functions can be called as many times as needed.
